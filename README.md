**Detection of Early Stage Parkinson’s Disease Using Wearable Sensors at Multiple Body Locations and Convolutional Neural Networks**


This repository contains the analysis code used in the manuscript:
“Detection of Early Stage Parkinson’s Disease Using Wearable Sensors at Multiple Body Locations and Convolutional Neural Networks”.

The repository includes code for preprocessing, recurrence plot generation, deep-learning model training and testing, machine-learning analyses, LASSO-based feature selection, Leave-One-Subject-Out validation, bootstrap confidence interval estimation, and more- and less-affected side sensitivity analysis.

Raw participant data are not included because of ethical and privacy restrictions.
